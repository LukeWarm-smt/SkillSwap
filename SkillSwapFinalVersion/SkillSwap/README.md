# Java Project: Skill Swap

## Folder Structure

The workspace contains two folders by default, where:

- `src`: the folder to maintain sources

Inside the `src`, there are four folders:

- `domain`: the folder where all the enums anc classes relevant to the student creation reside.
- `app`: the folder where the main is.
- `service`: the folder where the core logic of the program is.
- `state`: the foldr where all Maps are conatined
- `storage`: where all progress can be saved once the program ends.

## Compiling and Executing

### To compile
    javac SkillSwap/src/domain/*.java
    javac SkillSwap/src/app/*.java
    javac SkillSwap/src/service/*.java
    javac SkillSwap/src/state/*.java
    javac SkillSwap/src/storage/*.java

    or

    javac SkillSwap/src/domain/*.java SkillSwap/src/app/*.java SkillSwap/src/service/*.java SkillSwap/src/state/*.java
### To execute
    java SkillSwap/src/app/SkillSwap.java

## To delete classes
    rm -f SkillSwap/src/domain/*.class SkillSwap/src/app/*.class SkillSwap/src/service/*.class SkillSwap/src/state/*.class

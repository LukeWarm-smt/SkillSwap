## Folder Structure

The workspace contains two folders by default, where:

- `src`: the folder to maintain sources

Meanwhile, the compiled output files will be generated in the `bin` folder by default.

Inside the `src`, there are four folders:

- `domain`: the folder where all the enums anc classes relevant to the student creation reside.
- `app`
- `service`
- `storage`

## Compiling and Executing

### To compile
    javac SkillSwap/src/domain/*.java
    javac SkillSwap/src/app/*.java
    javac SkillSwap/src/domain/*.java SkillSwap/src/app/*.java SkillSwap/src/service/*.java SkillSwap/src/state/*.java
### To execute
    java SkillSwap/src/app/SkillSwap.java

## To delete classes
    rm -f SkillSwap/src/domain/*.class
    rm -f SkillSwap/src/app/*.class

The `JAVA PROJECTS` view allows you to manage your dependencies. More details can be found [here](https://github.com/microsoft/vscode-java-dependency#manage-dependencies).
